🧠 What This Dashboard Does

This isn’t just charts—it’s a full-on superhero analytics battleground.

The dashboard analyzes:

💰 Box office performance to see who truly rules financially
📈 Return on Investment (ROI) to measure efficiency, not just scale
⭐ Audience reception through IMDb and Rotten Tomatoes scores
🎭 Genre trends to uncover what actually works
🦸 Team-up vs solo films to see if Avengers-style collabs really pay off
📊 Year-wise trends showing how each universe evolved over time
