**Cinematic Clash** is an interactive data analytics dashboard that compares the performance of the Marvel Cinematic Universe and the DC Extended Universe. The project analyzes key factors such as box office revenue, return on investment, audience ratings, genre trends, and the impact of team-up versus solo films to uncover patterns in success and consistency across both universes. Through interactive visualizations, the dashboard enables users to explore how each franchise has evolved over time and what strategic elements contribute to their performance.
